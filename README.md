MathLibrary представляет собой легковесную библиотеку классов, предоставляющую набор основных математических операций и алгоритмов. Библиотека разработана для использования в .NET приложениях и не требует дополнительных зависимостей.

Возможности

\- Базовые арифметические операции (сложение, вычитание, умножение, деление)

\- Проверка чисел на простоту

\- Возведение в степень

\- Вычисление факториала

\- Решение квадратных уравнений
